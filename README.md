# Work Day Scheduler

## Description

A simple calendar application that allows a user to save events for each hour of the day. This app will run in the browser and feature dynamically updated HTML and CSS powered by jQuery.

## Technologies used

- Bootstrap
- JQuery
- Menent .JS
- HTML
- Javascript
- CSS

## Usage

- This app can run on any modern browser

- The app present timeblocks for standard business hours when the user scrolls down.

- The app contain Color-code each timeblock based on past, present, and future

- The app will allow a user to enter an event when they click a timeblock

- The app will allow user to save the event in local storage when the save button is clicked in that timeblock.

- The app will display a warning message when no input is entered on a time block and the save button is clicked
